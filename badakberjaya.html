Description

This application is one of the systems from the East Lampung Police which features Police Reports and Loss Reports, Information and community reporting via emergency number 110 etc.

this project was done in 10 days with 4 people and me as an Android Developer This system uses :

  • Kotlin as the programming language
  • Xml as the layout
  • Retrofit as the RestApi library.
  • Captcha UI for verification
  • Login session and token from the server side for Security
  • Some features only use Webview
